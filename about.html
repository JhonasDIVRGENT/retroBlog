¡Hola! Soy un aprendiz de tecnología

Mi Historia Personal

¿Quién soy?

¡Bienvenido a mi diario personal de aprendizaje tecnológico! Soy alguien que decidió sumergirse en el fascinante mundo de la programación y el desarrollo de software.

Este blog es mi espacio personal donde documento mi viaje de aprendizaje, mis descubrimientos, frustraciones, y esos momentos "¡Eureka!" que hacen que todo valga la pena.

🎯 Mi Enfoque Actual

  • Java: Mi lenguaje principal. Cada día descubro algo nuevo sobre POO, la JVM, y el ecosistema Java.
  • JPA/Hibernate: La persistencia de datos ya no es un misterio. Las anotaciones son magia pura.
  • Servlets: Mi puerta de entrada al desarrollo web. Request-response nunca fue tan emocionante.
  • Diseño de Sistemas: Mi próximo gran desafío. Quiero entender cómo se construyen aplicaciones escalables.

🔐 Mi Pasión Secreta

Aunque mi enfoque principal está en el desarrollo, tengo una fascinación oculta por la ciberseguridad. Cada línea de código que escribo me hace pensar: "¿Cómo podría alguien explotar esto?" No es paranoia, ¡es curiosidad!

Creo firmemente que para construir aplicaciones seguras, primero debo entender cómo funcionan por dentro. Por eso estudio desarrollo web: para algún día poder protegerlo.

📚 Mi Filosofía de Aprendizaje

Aprendo mejor cuando escribo sobre lo que estoy estudiando. Este diario no es solo para mí, sino para cualquier persona que esté en un viaje similar. Si mis experiencias pueden ayudar a alguien más, ¡misión cumplida!

🛠️ Mi Setup Actual

  • IntelliJ IDEA Community (mi IDE favorito)
  • Apache Tomcat 10
  • MySQL para bases de datos
  • Git & GitHub para control de versiones
  • Maven para gestión de dependencias
